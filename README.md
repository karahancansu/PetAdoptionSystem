 🐾 Pet Adoption System

A simple and object-oriented **Java console application** for managing pets in an animal shelter.  
This project demonstrates core **OOP principles** and basic **design patterns** in a clean and understandable way. 💻✨

---

 🌟 Project Overview

The **Pet Adoption System** is designed to simulate a small shelter management application.  
It allows users to:

- 🐶 Add different types of pets
- 📋 List available pets
- 🍖 Feed pets
- 🏠 Adopt pets
- 📝 Keep care and adoption records

This project was developed to practice **Object-Oriented Programming** concepts in Java.

---

 🚀 Features

✅ Add pets to the system  
✅ Support for multiple pet types such as **Dog, Cat, and Bird**  
✅ Display pet information  
✅ Feed pets and keep care records  
✅ Adopt pets and store adoption history  
✅ Organized class structure for better readability and maintainability  

---
 🧠 OOP Concepts Used

This project includes the following Object-Oriented Programming concepts:

- **Encapsulation** 🔒  
- **Inheritance** 👨‍👩‍👧‍👦  
- **Polymorphism** 🎭  
- **Abstraction** 🧩  

---

 Design Patterns Used

🏭 Factory Pattern
Used to create pet objects in a more flexible and organized way.

🔐 Singleton Pattern
Used for shelter settings so that only one shared settings object exists in the system.
